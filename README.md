# Numerical-guessing-game - Игра: "Числовая Угадайка"

Спрашиваем у игрока, хочет ли он указать границы загадываемого числа (максимальное число) начинаем с единицы.  
Если пользователь не вводит число, то граница автоматически ставится на 100. (от 1 до 100 включительно)
Если вы указали число больше или меньше загаданного, вам об этом сообщается до тех пор, пока не угадаете.  
После победы, будет указано количество попыток.  
А после, предложено сыграть ещё)  
В случае отказа, игра попращается и завершит выполнение.  
  
Так-же есть проверка "на дурака" если ввести не ожидаемые значения ответов)  


Приятной игры)
